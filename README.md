## Step 1 : install python

go web

```shell
$python --version

```

## Step 2 : install pipenv

```shell
$pip install pipenv

```

## Step 3 : create Django project

```shell
$mkdir nameProject
$cd nameProject
$pipenv install django
$pipenv shell
$django-admin startproject nameProject .
```

## Step 4 : runsrver port 8000

```shell
$python manage.py runserver 8000
```
